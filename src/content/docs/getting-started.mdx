## Install

From your local checkout of `anthurium-ai/orca`:

```bash
./install.sh
agent-orca --version
```

## Initialize a repo

In the repo you want to work on:

```bash
agent-orca init --preset go   # or node|python
agent-orca check
```

## Orchestrate (manual coder)

```bash
agent-orca loop . --manual-coder-a --fresh

# watch
agent-orca show --run latest
agent-orca logs loop  --run latest -f
agent-orca logs critic --run latest -f

# accept and land locally
agent-orca approve --run latest
agent-orca promote --run latest

# cleanup
agent-orca clean --all
```

## Task queue (multiple workers on different tasks)

```bash
agent-orca queue enqueue "Add /healthz endpoint"
agent-orca queue enqueue "Refactor config loading"
agent-orca queue ls --active

# in separate terminals:
agent-orca work --worker coder-1
agent-orca work --worker coder-2
```

Each worker creates its own worktree, waits for you to commit, then runs validation to accept/reject.

Fast path for one-off work:

```bash
agent-orca do "Add DELETE /todos/{id} endpoint" --mode auto --role coder
```

Useful recovery and hygiene commands:

```bash
# inspect failures
agent-orca queue ls --failed

# retry one failed item
agent-orca queue retry <item_id>

# retry latest failed item
agent-orca queue retry --latest-failed

# retry all failed items (optionally cap count)
agent-orca queue retry --all-failed --limit 3

# remove stale worker worktrees
agent-orca queue gc
```

Evidence is written per task at:

```text
.agent-orca/runs/queue/<item_id>/evidence.json
```

And `agent-orca show --run latest` includes queue totals, active workers, and stale lease count.
