Orca is a **verifier-first agent orchestrator**.

- **Verification is the product**: `agent-orca check` runs your repoâ€™s definition of done from `orca.yaml`.
- **Orchestration is the control plane**: runs, roles/workers, logs, evidence, and safe promotion.

## The 60-second tour

```bash
# install (from your orca checkout)
./install.sh

# in any repo
agent-orca init --preset go
agent-orca check

# orchestrate (manual)
agent-orca loop . --manual-coder-a --fresh
agent-orca show --run latest
agent-orca logs critic --run latest -f
agent-orca approve --run latest
agent-orca promote --run latest
```
